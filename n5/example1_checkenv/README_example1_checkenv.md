# n5: 基礎環境檢查 (Check Env)

本目錄提供了在國網中心 **晶創主機 (n5 / Nano5)** 平台上進行系統配置、單點與多設備資源探索的基礎測試腳本。

這部分的核心設計理念（如：確認 `srun` 是否有抓到正確的 `$NGPU`、如何進入 Conda 環境等）完全與 TWCC 平台通用。為避免重複造輪子，這裡僅存放針對 n5 硬體環境 (H100、對應 partition) 調整過 `#SBATCH` 的腳本。

📌 **詳細的文件與教學，請直接參考 TWCC 平台的文件：**
👉 [../../twcc/README_twcc.md](../../twcc/README_twcc.md) (查閱其中的 **2. hello world** 與 **3. 用 hello 看世界** 章節)

你可以直接利用本目錄的腳本（例如 `check_env.sb`）送到 n5 節點測試，日誌將顯示分配給您的 H100 運算節點環境狀態。
